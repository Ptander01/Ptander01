# Valencia Building Age Maps — Cinematic Portfolio Upgrades

This collection transforms raw R/RStudio data visualizations of Valencia, Spain's building ages into cinematic, museum-quality art pieces. The original project explored the age of buildings within Valencia using data from INSPIRE, inspired by Dominic Roye's "Intro to GIS with R" blog post.

**Tools Used (Original):** R, RStudio, tmap, ggplot2, colorRampPalette, tmaptools (Geocode_OSM, st_buffer, st_transform, st_intersection)

**Data Source:** INSPIRE — downloaded via feed.extract(url), filtered by province

---

## Folder Structure

### 01_Hero_Wide_Shots
Polished wide-angle views of the full Valencia building age map, presented as museum gallery installations with dramatic lighting. The stained-glass aesthetic transforms building footprint data into luminous, jewel-toned mosaics.

### 02_Detail_Stained_Glass
Close-up detail shots emphasizing the cathedral stained-glass window aesthetic — with authentic lead came lines, translucent colored glass textures, and dramatic backlighting. Includes a Gothic rose window interpretation.

### 03_Macro_3D_Texture
Super close-up macro shots where the data is physically extruded as 3D organic materials:
- **Crystal/Glass** — Building footprints as translucent crystal blocks with prismatic light refraction
- **Brushed Metal** — Oxidized copper, brass, titanium, and chrome with authentic patina
- **Carved Wood** — Mahogany, walnut, cherry, maple, and ebony with natural grain patterns
- **Ceramic/Clay** — Hand-molded terracotta with raku and celadon glazes

### 04_Chart_and_Legend
Museum-quality supporting graphics:
- Evolution of Urban Development density chart (1756–2016) with stained-glass gradient fill
- Vertical color legend rendered as luminous stained-glass swatches
